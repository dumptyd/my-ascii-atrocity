# my-ascii-atrocity
Convert string to ascii art.
***
##Usage

`git clone https://github.com/dumptyd/my-ascii-atrocity.git`

`cd my-ascii-atrocity`

`node maa <text> <character to use for art> <character to use instead of space>`
***
##Demo

- `node maa hey #`

        ```
        #   # ##### #   #
        #   # #      # #
        ##### ####    #
        #   # #       #
        #   # #####   #
        ```

- `node maa hey :neckbeard: :w:`
